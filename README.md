# JavaScript Portfolio Website

A modern, interactive portfolio website showcasing JavaScript capabilities with live demos and examples.

## Features

- **Interactive Demos**: Five different JavaScript demonstrations including:
  - Counter Application
  - Text Transformer
  - Array Manipulator
  - Quote Generator
  - Background Color Changer

- **Modern Design**: Dark theme with JavaScript's signature yellow (#f7df1e) accent color
- **Responsive Layout**: Works on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in effects and hover interactions
- **No Dependencies**: Pure vanilla JavaScript, HTML, and CSS

##  Project Structure

##  Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.) if you want to modify the code

### Installation

1. Download all three files:
   - `index.html`
   - `styles.css`
   - `script.js`

2. Place them in the same folder

3. Open `index.html` in your web browser

That's it! No build process or installation required.

##  Demo Explanations

### Counter Application
Demonstrates state management and DOM manipulation. Increment, decrement, or reset a number.

### Text Transformer
Shows string manipulation methods including:
- Converting to uppercase/lowercase
- Reversing text
- Counting words and characters

### Array Manipulator
Illustrates array methods:
- Sorting numbers
- Reversing order
- Adding random elements
- Filtering even numbers

### Quote Generator
Randomly selects and displays programming quotes from an array.

### Background Color Changer
Generates random RGB colors and applies them to the page background.
